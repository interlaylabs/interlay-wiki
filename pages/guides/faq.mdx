# FAQ

Frequently asked questions about Interlay AI.

## General

### What is Interlay AI?
Interlay is a platform for seamlessly integrating your digital life with AI automation.

### How does it work?
Interlay combines a Memory Bank for data storage with Automation Flows for workflow execution.

### Is my data secure?
Yes, all data is encrypted and isolated by tenant with role-based access control.

## Development

### What tech stack is used?
Next.js, TypeScript, PostgreSQL, Temporal, Kafka, and more.

### How do I add a new integration?
Create OAuth flow, add to Vault, create trigger nodes.

### How do I debug workflows?
Use Temporal Web UI and execution logs.
