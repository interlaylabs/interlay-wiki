# Integrations

Connect Interlay AI with your favorite services and platforms.

## Available Integrations

### Gmail
Full email integration with Google Workspace:
- OAuth 2.0 authentication
- Real-time email triggers via PubSub
- Send and receive emails
- Attachment handling

### Discord
Bot-based Discord integration:
- Message monitoring
- Command handling
- Channel management
- Role-based permissions

### Clerk
Authentication and user management:
- SSO support
- Multi-tenant isolation
- Role-based access control
- Webhook events

### Google Calendar
Calendar synchronization:
- Event triggers
- Meeting preparation
- Scheduling automation
- Attendee management

### Stripe
Payment processing:
- Subscription management
- Usage-based billing
- Invoice generation
- Webhook handling
