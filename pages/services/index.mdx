# Services

Microservices architecture powering Interlay AI's distributed system.

## Service Architecture

### Event Processing Service
Handles external triggers and event routing:
- Kafka event streaming
- Webhook processing
- Trigger management
- Event transformation

### Flow Orchestration Service
Executes automation workflows:
- Temporal workflow engine
- Node execution
- Retry logic
- Error handling

### HashiCorp Vault Service
Secure secrets management:
- OAuth token storage
- API key management
- Certificate handling
- Encryption services

## Deployment

All services are containerized and deployed via Railway with:
- Automatic scaling
- Health checks
- Log aggregation
- Monitoring
